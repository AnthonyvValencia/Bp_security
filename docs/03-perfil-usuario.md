# Perfil del Usuario — BP Security

## Campos requeridos

- Nombres.
- Apellidos.
- Correo electrónico.
- Teléfono.
- Dirección.
- Barrio.
- Número de casa.
- Referencias del domicilio.
- Fotografía.
- Contactos de emergencia.

## Campos adicionales

> Se puede proponer información adicional si aporta valor al proyecto
> (por ejemplo, tipo de sangre, condiciones médicas relevantes para
> emergencias, etc.), siempre justificando por qué se agrega antes de
> implementarla.

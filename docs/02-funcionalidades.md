# Funcionalidades — BP Security

## Funcionalidades principales (mínimo requerido)

- Registro.
- Inicio de sesión.
- Recuperación de contraseña.
- Perfil.
- Gestión de comunidades.
- Gestión de miembros.
- Botón de pánico.
- Chat comunitario.
- Compartir ubicación GPS.
- Llamada al ECU 911.
- Reportes.
- Historial de reportes.
- Historial de alertas.
- Notificaciones Push.
- Sonido de emergencia.
- Configuración.
- Cierre de sesión.
